Librariux


Description: Librariux is an intelligent book recommendation system designed to streamline the reading experience by automating personalized book suggestions. The system schedules and sends curated book recommendations via email at user-specified intervals, complete with detailed summaries and direct download links. This eliminates the need for manual searching, making the process seamless and convenient. The project leverages Python, Pandas, and the SMTP library to efficiently manage data and handle email communication.
